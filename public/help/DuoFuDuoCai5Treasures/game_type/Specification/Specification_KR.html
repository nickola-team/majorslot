게임규칙

1.배팅 레벨은 배팅 레벨 지정을 사용하여 설정됩니다.

2.코인 값은 코인 값 선택자를 사용하여 설정됩니다.

3.코인은 배팅 할 수있는 코인의 수를 표시합니다.

4.자동플레이는 선택한 라운드 수만큼 게임을 자동으로 실행합니다.

5.와일드 기호는 보너스 또는 무료 스핀 기호를 제외한 모든 기호를 대신합니다.

6.조합에 따라 승점 조합 및 상금 지급이 이루어집니다.

7.배팅레벨은 배팅 라인 당 배팅 한 코인의 수입니다.

8.코인에서의 배팅 라인 우승은 페이테이블에 표시된 값과 배팅 레벨을 곱한 값과 같습니다.

9.배팅 라인이 통화로 승리하는 것은 코인의 승점과 코인값을 곱한 것과 같습니다.

10.배팅 라인 당 최고 득점수만 지급됩니다.

11.다른 배팅 라인의 동시 승점이 추가됩니다.

12.모든 코인 우승은 보너스 게임의 승점을 제외하고는 배팅 라인에서만 지불합니다.

빠른정지 : 스핀이 멈추는 동안 게임 영역을 누릅니다.

소리

자동플레이

설정 메뉴로 이동하지 않고 자동 실행을 사용 또는 사용 중지하려면 탭하세요. 기본적으로 50 라운드가 설정됩니다.

빠른스핀

탭하여 설정 메뉴로 이동하지 않고 빠른 스핀을 켜거나 끕니다.

게임 설정 및 게임 정보

소리 설정

게임 음악 : 스위치를 살짝 눌러 소리를 켜거나 끕니다.

게임 소리 : 스위치를 살짝 눌러 소리를 켜거나 끕니다.

스핀설정

자동플레이:슬라이더를 드래그하거나 살짝 눌러 자동 재생을 활성화하면 자동 재생 옵션과 라운드 수를 설정할 수 있습니다. 자동 재생은 선택한 횟수만큼 회전합니다.

참고 : 게임 도중 연결이 끊어지면 게임을 다시로드 할 때 모든 자동 실행 설정이 기본값으로 돌아갑니다.

참고 : 일부 자동 재생 옵션은 일부 관할 지역에서 필수입니다.

빠른스핀 : 빠른 스핀을 켜거나 끌 때 스위치를 가볍게 누르십시오.

왼손 모드 : 왼쪽 버튼을 켜거나 끌 때를 살짝 누르십시오.

배팅설정

현금배팅 : 선택한 통화로 귀하의 배팅 금액을 보여줍니다.

배팅레벨 : 슬라이더를 드래그하거나 탭하여 배팅 라인 당 배팅 할 코인 수를 선택하십시오.

코인값 : 슬라이더를 드래그하거나 탭하여 코인 값을 선택하십시오.
